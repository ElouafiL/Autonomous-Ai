## ✨ الميزات

- 🧠 **8 وكلاء متخصصين**: مبتكر، محلل، مبرمج، باحث، منفذ، خبير منطقي، سريع، وخبير رسوم
- 🔗 **Fallback Chain تلقائي**: إذا فشل نموذج، ينتقل النظام تلقائياً لبديل
- 🧰 **أدوات خارجية**: آلة حاسبة، بحث حقيقي، وقت، ملاحظات، ورسوم بيانية
- 🧠 **ذاكرة ديناميكية**: قصيرة المدى (سياق) + طويلة المدى (دروس مستفادة)
- 🎯 **إدراك فوق معرفي**: يراقب الأداء ويوقف الجدال العقيم تلقائياً
- 🗳️ **نظام تصويت**: الوكلاء يصوتون لاختيار أفضل إجابة
- 🔧 **تفكير مع نقد ذاتي**: `think_with_reflection` لتحسين الجودة
- 💬 **واجهة عربية كاملة**: دعم كامل للغة العربية في المدخلات والمخرجات
