### **1. فهم الخوارزمية**
**الفرز بالتصاعد (Bubble Sort)** هي خوارزمية فرز بسيطة تعتمد على:
- **مقارنة العناصر المتجاورة** في القائمة وتبديلها إذا كانت في الترتيب الخاطئ.
- **تكرار العملية** حتى لا تحدث أي تبديلات (القائمة تصبح مرتبة).
- **ميزة الاستقرار**: تحافظ على ترتيب العناصر المتساوية.

---

### **2. خطوات الخوارزمية**
1. ابدأ من أول عنصر في القائمة.
2. قارن العنصر الحالي مع العنصر التالي.
3. إذا كان العنصر الحالي أكبر من التالي، **بدّل بينهما**.
4. انتقل إلى العنصر التالي وكرر المقارنة حتى نهاية القائمة.
5. **كرر العملية** للقائمة حتى تكمل دورة كاملة **دون أي تبديلات** (القائمة مرتبة).

---

### **3. التعقيد الزمني والمكاني**
| الحالة          | التعقيد الزمني | الشرح                                                                 |
|-----------------|------------------|------------------------------------------------------------------------|
| **أسوأ حالة**   | O(n²)            | القائمة مرتبة بالعكس، نحتاج لـ `n` دورات وكل دورة `n` مقارنات.      |
| **أفضل حالة**   | O(n)             | القائمة مرتبة أصلًا **مع تحسين `swapped`** (دورة واحدة فقط).        |
|                 | O(n²)            | القائمة مرتبة أصلًا **بدون تحسين** (تكمل جميع الدورات).             |
| **متوسط حالة**  | O(n²)            | في معظم الحالات، نحتاج لـ `n²/2` مقارنة.                             |
| **التعقيد المكاني** | O(1)         | لا تحتاج لمساحة إضافية (فرز في المكان).                              |

### **4. مثال عملي بلغة بايثون**
```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False  # تحسين الأداء: الخروج مبكرًا إذا كانت القائمة مرتبة
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]  # تبديل العناصر
                swapped = True
        if not swapped:  # إذا لم يحدث أي تبديل، القائمة مرتبة
            break
    return arr

# مثال الاستخدام
numbers = [64, 34, 25, 12, 22, 11, 90]
sorted_numbers = bubble_sort(numbers)
print("القائمة المرتبة:", sorted_numbers)  # الناتج: [11, 12, 22, 25, 34, 64, 90]
```

---

### **5. شرح الكود**
1. **الحلقة الخارجية (`for i in range(n)`)**:
   - تضمن مرورًا كاملًا على جميع العناصر.
2. **الحلقة الداخلية (`for j in range(0, n-i-1)`)**:
   - تقارن العناصر المتجاورة وتبدلها إذا لزم الأمر.
   - تقلل المدى مع كل دورة لأن أكبر العناصر "يطفو" إلى النهاية.
3. **متغير `swapped`**:
   - يحسن الأداء بالخروج مبكرًا إذا كانت القائمة مرتبة (أفضل حالة `O(n)`).
4. **التبديل (`arr[j], arr[j+1] = arr[j+1], arr[j]`)**:
   - يبدل بين العنصرين باستخدام تقنية بايثون المبسطة.

1. **اختيار الخوارزمية:** اختر خوارزمية فرز بسيطة مثل "الفرز بالتصاعد" (Bubble Sort) أو "الفرز السريع" (Quick Sort). سأختار الفرز بالتصاعد لأنه بسيط وسهل الفهم.

2. **شرح الخوارزمية:**
   - **الفرز بالتصاعد (Bubble Sort):** هذه الخوارزمية تعمل بفرز القائمة عن طريق مقارنة كل زوج من الأرقام المجاورة وتبادلها إذا كانت ترتيبها غير صحيح. تكرار هذه العملية يؤدي إلى فرز القائمة بالكامل.

3. **التعقيد الزمني:**
   - **أفضل حالة:** O(n) cuando تكون القائمة مفروسة بالفعل.
   - **أسوأ حالة:** O(n^2) عندما تكون القائمة غير مفروسة على الإطلاق (مثل فرز تنازلي).
   - **الحالة المتوسطة:** O(n^2) لأن الفرز بالتصاعد يعتبر غير كفء للقوائم الكبيرة.

4. **مثال كود بايثون:**
   ```python
   def bubble_sort(arr):
       n = len(arr)
       for i in range(n):
           # tạo một vòng lặp để so sánh các phần tử
           for j in range(0, n - i - 1):
               # nếu phần tử trước lớn hơn phần tử sau thì đổi chỗ
               if arr[j] > arr[j + 1]:
                   arr[j], arr[j + 1] = arr[j + 1], arr[j]
       return arr

   # Ví dụ sử dụng
   numbers = [64, 34, 25, 12, 22, 11, 90]
   print("Trước khi sắp xếp:", numbers)
   print("Sau khi sắp xếp:", bubble_sort(numbers))
   ```

**النتائج:**
- الخوارزمية المختارة هي الفرز بالتصاعد.
- التعقيد الزمني للفرز بالتصاعد هو O(n^2) في أسوأ الحالات.
- قدّم مثال كود بايثون لتنفيذ الخوارزمية.

**الاستنتاج:**
الفرز بالتصاعد是一种 خوارزمية بسيطة لفرز القوائم، لكنها قد تكون غير كفئة للقوائم الكبيرة بسبب تعقيد زمنيها O(n^2). ومع ذلك، فهي مفيدة للفهم الأساسي لأسلوب الفرز والخوارزميات.
